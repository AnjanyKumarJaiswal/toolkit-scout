What I built

1 · Research pipeline

Python + Flask. One search-grounded gpt-4o-mini call per app (Composio's composio_search toolkit when configured, OpenAI's built-in web_search otherwise), strict JSON-schema output, resumable JSON store, per-app cost logging. Full run: 100/100 apps, ~$1.16. Re-runnable via POST /api/run.

Agent did this

2 · Verification loops

Random sample of 18 apps, two loops: a strict fact-checker agent re-judges every claim against the fetched evidence page, then flagged fields get a literal fetch-and-confirm pass — a correction only counts if its supporting quote appears verbatim on the page. A human review UI covers the manual check.

Agent + human

3 · Pattern analysis

Aggregates computed over all structured records: auth and access-model distributions, verdicts per category, blocker frequency. Exported as static JSON for this page.
